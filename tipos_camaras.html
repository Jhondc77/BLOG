Cámaras de Seguridad IP (Internet Protocol):

Las cámaras de seguridad IP son aquellas que utilizan un protocolo IP (Internet Protocol) para mostrar las imágenes que graban. Así, las imágenes pueden guardarse y es posible tener acceso a ellas a través de la red. Estas Cámaras suelen ser más baratas que, por ejemplo, las cámaras CCTV (Circuito Cerrado de Televisión). Las cámaras IP se conectan a un grabador de vídeo y a un router a través de un cable UTP o de manera inalámbrica (por Wifi). Así, las imágenes pueden almacenarse y es posible acceder a ellas a través de la red.


Caracteristicas

* Estas cámaras utilizan una conexión de red para transmitir imágenes y datos a través de Internet.

*Pueden ser cableadas o inalámbricas.

*Ofrecen alta resolución de video y funcionalidades avanzadas como detección de movimiento, visión nocturna y zoom digital.

Cámaras de Seguridad PTZ (Pan-Tilt-Zoom)

Las cámaras PTZ son las cámaras más versátiles disponibles. Son perfectos para muchas aplicaciones diferentes. Cambie su imagen de una descripción general de la escena a un detalle específico. Gire su vista a diferentes ángulos de forma remota. Todas estas características hacen que una cámara PTZ sea una cámara todo en uno. Puede reemplazar un sistema de múltiples cámaras actual de manera más eficiente. En lugar de tener que usar numerosas cámaras fijas, ahora puede usar solo algunas cámaras PTZ y aún así poder capturar más del momento. Nuestras cámaras Avonic PTZ están diseñadas para instalaciones fijas y funcionan muy bien en condiciones de poca luz. Puede controlar la cámara con protocolos comunes: VISCA, Pelco-D/P a través de RS232, RS485, IR y, lo más importante, sobre IP. Nuestras cámaras Avonic están equipadas con un control remoto IR, pero también se pueden controlar con nuestro controladores.


Caracteristicas

*Permiten el control remoto de la dirección de la cámara (paneo y inclinación) y el zoom para obtener una vista más detallada de una escena.

*Son ideales para áreas grandes o donde se requiere una vigilancia dinámica.

Cámaras de Seguridad PTZ (Pan-Tilt-Zoom)

Las cámaras de vigilancia PoE son cámaras IP (también llamadas cámaras de red) que utilizan la tecnología PoE y que ofrecen tanto datos como alimentación eléctrica mediante un solo cable. Tenga en cuenta que aunque todas las cámaras PoE son cámaras IP, no todas las cámaras IP son cámaras PoE. A diferencia de las cámaras CCTV analógicas tradicionales, las cámaras IP (y las cámaras PoE) se pueden utilizar como un dispositivo independiente. Para conectar cámaras analógicas a la red, estas cámaras requieren una DVR con el fin de convertir la señal analógica a IP.


Caracteristicas

*Estas cámaras reciben alimentación y transmiten datos a través de un solo cable Ethernet, lo que simplifica la instalación y reduce la necesidad de cables adicionales.
